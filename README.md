# rekey
A keyboard remap tool

# How to build

# Requirements
- Ruby (see [.ruby-version](./.ruby-version) for strict version)
- Xcode 11

# Setup
```console
bundle install
pod install
```

# Lint
```console
Pods/SwiftLint/swiftlint
```

# Format
```console
Pods/SwiftFormat/CommandLineTool/swiftformat .
```

# Run
Open `rekey.xcworkspace` with Xcode and run.
